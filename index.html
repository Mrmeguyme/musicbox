Installation

Libraries / Python

The following libraries are required for the music box:

  • Mutagen

  • Pygame

These libraries can be installed through a few simple commands.

pip install mutagen

pip install pygame

These commands require python, Which can be downloaded from https://www.python.org/downloads/. Make sure Python 3 is downloaded, as it is required for the program to run correctly.

Program Time

Next, download the program, and extract it to a folder of your choosing. Run either setup.bat or setup.sh depending on whether you run a windows machine or a linux machine. This will prompt you to press any key, which when pressed will execute the program.

When executing the program from now on, execute it from run.bat.

Troubleshooting

if the program won't start, try typing

python Main.py build

into your command line to build the program.
